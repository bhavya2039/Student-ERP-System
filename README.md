# Student ERP System (Python, OOP)

A console-based Student ERP (Enrollment & Records Program) built using Core Python and Object-Oriented Programming. The project manages student records, academics, course & semester information, searching, sorting, and statistics through a menu-driven interface without using external libraries.

---

## Features

### Student Management
- Add Student
- View All Students
- Search Student
- Update Student
- Delete Student (Soft Delete)
- View Deleted Students

### Academics
- Add Marks
- Update Marks
- View Marks
- Calculate Total Marks
- Calculate Percentage
- Find Topper
- Find Lowest Marks
- Calculate Average Marks

### Course & Semester
- Assign Course
- Update Course
- Assign Semester
- View Students by Course
- View Students by Semester

### Statistics
- Count Total Students
- Count Students by Course
- Count Students by Semester

### Sorting
- Sort by Name
- Sort by Marks
- Sort by Percentage

### Searching
- Search by Roll Number
- Search by Name
- Search by Course

### Reports
- Display Student Details
- Display Top 3 Students
- Display Failed Students
